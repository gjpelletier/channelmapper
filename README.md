'channelmapper' is a Python toolset for mapping and analyzing meandering channels in satellite imagery. It accompanies the following paper: "Autogenic translation and counter point bar deposition in meandering rivers", by Z. Sylvester, P.R. Durkin, S.M. Hubbard, and D. Mohrig, GSA Bulletin, v.133, DOI:10.1130/B35829.1.
